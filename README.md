# AI・IT利用ガイドライン ジェネレーター v1.0.0

> 規模5パターン × 業種11カテゴリで、自社にあわせたAI・IT利用ガイドラインを生成するツール

---

## 📦 このツールでできること

5ステップの入力で、貴社にあわせた **4つの成果物** を生成します：

1. **AI・IT利用ガイドライン本体** ─ 全従業員向けの基本ルール集
2. **自社カスタマイズ運用ガイドブック** ─ 業種・規模にあわせて作り変える運用書
3. **生成AIリスク対策ハンドブック** ─ リスクマネジメント視点の打ち手集
4. **機密情報マスキング 実務マニュアル** ─ 現場ですぐ使える実務手引き

---

## 🚀 使い方

ブラウザで `index.html` を開いて、5ステップに入力するだけ：

1. **STEP 1：基本情報** ─ 会社名・業種・従業員数 → 規模区分（S/XS/S2/M/L）自動判定
2. **STEP 2：業種選択** ─ 11業種カテゴリから該当業種を複数選択
3. **STEP 3：AI利用ヒアリング** ─ 利用ツール／用途／取扱データ／マスキング場面
4. **STEP 4：既存規程** ─ 既存社内規程の有無
5. **STEP 5：出力形式** ─ HTML / Word / PDF / .md / 専門家向け要件定義書

入力途中で保存できます。「📥 .md保存」で途中状態をダウンロードし、後日「📂 .md読込」で続きから再開可能。

---

## 📤 出力形式

| 形式 | 主用途 |
|---|---|
| HTML | ブラウザですぐ読める。社内ポータル掲載に最適 |
| Word（.docx） | 社内で改訂・追記したい場合 |
| PDF | 印刷配布・読み専用 |
| .md | 後日続きから編集再開する用 |
| 専門家向け 要件定義書.md | 完全カスタマイズ版（弁護士確認込み）のご相談用 |

## 🛡️ 法務的位置づけ

本ジェネレーターは **雛形提供ツール** であり、**法務的助言の代替ではありません**。

実際に社内規程として施行する前に、必ず **弁護士・社労士の最終確認** をお受けください。

弁護士確認の目安：
- 個人事業主・〜10名規模：スポット相談（3〜5万円程度）
- 11〜100名規模：レビュー（10〜20万円程度）
- 101名〜規模：業界専門弁護士による本格レビュー

特に **医療・金融・士業** の各業界では、業界専門弁護士による個別確認を強く推奨します。
